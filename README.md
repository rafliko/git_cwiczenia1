# Projekt kryptograficzny

program zawiera:
  - Szyfr Vigenera
    - Szyfr Vigenera to szyfr polialfabetyczny, w którym każda litera tekstu jest przsuwana o różną liczbę pozycji zależną od powtarzanego hasła (klucz).
  - Moduł do łamania szyfru Vigenere

## Funkcjonalności
1. Szyfrowanie i deszfrowanie teskstu przy użyciu szyfru Vigenera
2. Generowanie losowych kluczy
3. Próba łamania zaszyfrowanych wiadomości
